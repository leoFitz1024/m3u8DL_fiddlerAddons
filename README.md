# m3u8DL_fiddlerAddons
一个fiddler插件，可以通过正则表达式匹配链接，配合m3u8DL进行视频下载

## 使用方法
下载代码编译后将项目obj/deBug目录下的FiddlerExtension.dll放入fiddler安装目录的Scripts文件加下即可。
打开fiddler就能看到插件tab。我用的是VS Studio2019
